## Cours 1 :  Introduction, Modèle Entités-Associations
#       Base de données (BD): ensemble structuré de données types, représentant une réalité spécifique (inventaire,réseau social)
#       Architecture Client-Serveur:
           Le **Serveur** est une instance d'un **SGBD** (Système de Gestion de BD) qui contrôle l'accès à la BD, les **Clients** sont les **programmes/applications** (Python,Java...) qui communiquent avec le **Serveur**, et la **Communication** est un échange de **Requêtes** base de données (SQL,protocole JDBC)

#       Serveur SGBD:
            - Reçoit **Requêtes** de tout les **clients** (applications), ordonne, optimise et exécute les requêtes reçues, gère les conflits d'accès, les pannes et les requêtes eronnées, et renvoie la réponse aux **clients**
